# Netlify-তে OneChain DeFi Platform Deploy করার সহজ গাইড

**লেখক: Manus AI**

আপনি জানতে চেয়েছেন কীভাবে Netlify-তে এই প্রজেক্টটি deploy করবেন। এটা খুব ভালো প্রশ্ন, কিন্তু এখানে কিছু বিষয় আছে যা আপনাকে আগে বুঝতে হবে।

## Netlify কী এবং এই প্রজেক্টের সাথে এর সম্পর্ক কী?

Netlify হলো একটি প্ল্যাটফর্ম যা মূলত **স্ট্যাটিক ওয়েবসাইট (Static Websites)** এবং **ফ্রন্টএন্ড অ্যাপ্লিকেশন (Frontend Applications)** হোস্ট করার জন্য খুব জনপ্রিয় [1]। স্ট্যাটিক ওয়েবসাইট মানে হলো এমন ওয়েবসাইট যেখানে HTML, CSS, JavaScript ফাইল থাকে এবং এগুলো কোনো ডেটাবেস বা সার্ভার (backend) ছাড়াই সরাসরি ব্রাউজারে চলে।

আপনার `onechain-defi-platform` প্রজেক্টটি কিন্তু শুধু স্ট্যাটিক ওয়েবসাইট নয়। এর তিনটি প্রধান অংশ আছে:

1.  **ফ্রন্টএন্ড (Frontend):** `client` ফোল্ডারের ভিতরে আছে। এটা React দিয়ে বানানো এবং ব্রাউজারে চলে।
2.  **ব্যাকএন্ড (Backend):** `server` ফোল্ডারের ভিতরে আছে। এটা Node.js এবং Express দিয়ে বানানো এবং সার্ভারে চলে।
3.  **ডেটাবেস (Database):** এটা MySQL ব্যবহার করে, যেখানে সব তথ্য (যেমন ইউজার, ব্যালেন্স) জমা থাকে।

**সমস্যাটা হলো:** Netlify সরাসরি Node.js ব্যাকএন্ড সার্ভার বা MySQL ডেটাবেস হোস্ট করে না। Netlify-তে আপনি শুধু ফ্রন্টএন্ড অংশটি (যা `client` ফোল্ডারে আছে) deploy করতে পারবেন। ব্যাকএন্ড এবং ডেটাবেসের জন্য আপনাকে অন্য কোনো সার্ভিস ব্যবহার করতে হবে।

> **গুরুত্বপূর্ণ:** যদি আপনি শুধু এই প্রজেক্টের ল্যান্ডিং পেজ বা ফ্রন্টএন্ড ডিজাইন দেখাতে চান, তাহলে Netlify ব্যবহার করা ঠিক আছে। কিন্তু যদি আপনি লগইন, ডেটাবেস অপারেশন বা অন্য কোনো সার্ভার-সাইড ফাংশনালিটি চালাতে চান, তাহলে শুধু Netlify-তে deploy করলে হবে না। তখন আপনার ব্যাকএন্ড এবং ডেটাবেস অন্য কোথাও হোস্ট করতে হবে এবং ফ্রন্টএেন্ডকে সেই ব্যাকএন্ডের সাথে কানেক্ট করতে হবে।

এখন আমি আপনাকে দেখাবো কীভাবে এই প্রজেক্টের **ফ্রন্টএন্ড অংশটি** Netlify-তে deploy করবেন।

## ধাপ ১: প্রজেক্টের ফ্রন্টএন্ড বিল্ড করুন

Netlify-তে deploy করার আগে, আপনার প্রজেক্টের ফ্রন্টএন্ড অংশটিকে 
বিল্ড করতে হবে। বিল্ড করার মানে হলো, React কোডগুলোকে এমন ফাইলে পরিণত করা যা ব্রাউজার সরাসরি বুঝতে পারে (HTML, CSS, JavaScript)।

*   আপনার Command Prompt/Terminal ওপেন করুন এবং আপনার `onechain-defi-platform` ফোল্ডারে যান (যদি না থাকেন)।

    ```bash
    cd path/to/onechain-defi-platform
    ```

*   এবার নিচের কমান্ডটি টাইপ করে **Enter** চাপুন:

    ```bash
    pnpm run build
    ```

*   এই কমান্ডটি `client` ফোল্ডারের React কোডগুলোকে বিল্ড করবে এবং `dist` নামে একটি নতুন ফোল্ডার তৈরি করবে। এই `dist` ফোল্ডারের ভিতরেই আপনার ওয়েবসাইটের সব স্ট্যাটিক ফাইল থাকবে যা Netlify-তে আপলোড করতে হবে।

## ধাপ ২: Netlify-তে অ্যাকাউন্ট তৈরি করুন এবং লগইন করুন

*   যদি আপনার Netlify অ্যাকাউন্ট না থাকে, তাহলে [https://www.netlify.com/](https://www.netlify.com/) ওয়েবসাইটে যান এবং একটি ফ্রি অ্যাকাউন্ট তৈরি করুন। আপনি GitHub, GitLab বা ইমেইল দিয়ে সাইন আপ করতে পারেন।
*   অ্যাকাউন্ট তৈরি হয়ে গেলে বা যদি আপনার অ্যাকাউন্ট থাকে, তাহলে লগইন করুন।

## ধাপ ৩: Netlify-তে প্রজেক্টটি আপলোড করুন

Netlify-তে প্রজেক্ট আপলোড করার দুটি প্রধান উপায় আছে:

### উপায় ১: GitHub/GitLab/Bitbucket থেকে কানেক্ট করা (সবচেয়ে ভালো উপায়)

যদি আপনার প্রজেক্টটি GitHub, GitLab বা Bitbucket-এ থাকে, তাহলে এই পদ্ধতিটি সবচেয়ে সহজ এবং স্বয়ংক্রিয়।

1.  Netlify ড্যাশবোর্ডে লগইন করার পর, **Add new site** বাটনে ক্লিক করুন।
2.  **Import an existing project** সিলেক্ট করুন।
3.  আপনার Git প্রোভাইডার (যেমন GitHub) সিলেক্ট করুন এবং আপনার অ্যাকাউন্ট কানেক্ট করুন।
4.  আপনার `onechain-defi-platform` প্রজেক্টের রিপোজিটরি (repository) খুঁজে বের করে সিলেক্ট করুন।
5.  **Deploy settings** পেজে আপনাকে কিছু তথ্য দিতে হবে:
    *   **Owner:** আপনার Git অ্যাকাউন্ট।
    *   **Branch to deploy:** `main` বা `master` (আপনার প্রজেক্টের প্রধান ব্রাঞ্চ)।
    *   **Base directory:** এখানে `.` (একটি ডট) দিন, কারণ `package.json` প্রজেক্টের রুট ফোল্ডারে আছে।
    *   **Build command:** `pnpm run build` (যদি আপনি `pnpm` ব্যবহার করেন) অথবা `npm run build` (যদি `npm` ব্যবহার করেন)।
    *   **Publish directory:** `dist` (কারণ `pnpm run build` কমান্ডটি `dist` ফোল্ডারে ফাইল তৈরি করে)।
6.  সবকিছু ঠিক থাকলে, **Deploy site** বাটনে ক্লিক করুন।
7.  Netlify আপনার প্রজেক্টটি বিল্ড করা শুরু করবে এবং সফলভাবে deploy হলে আপনাকে একটি লাইভ URL দেবে।

### উপায় ২: ম্যানুয়ালি `dist` ফোল্ডার আপলোড করা (সহজ কিন্তু স্বয়ংক্রিয় নয়)

যদি আপনার প্রজেক্ট Git-এ না থাকে বা আপনি দ্রুত শুধু `dist` ফোল্ডারটি আপলোড করতে চান, তাহলে এই পদ্ধতি ব্যবহার করতে পারেন।

1.  **ধাপ ১** অনুযায়ী `pnpm run build` কমান্ড চালিয়ে `dist` ফোল্ডারটি তৈরি করুন।
2.  Netlify ড্যাশবোর্ডে লগইন করার পর, **Add new site** বাটনে ক্লিক করুন।
3.  **Deploy manually** অপশনটি সিলেক্ট করুন।
4.  আপনার কম্পিউটারে `onechain-defi-platform` ফোল্ডারের ভিতরে থাকা `dist` ফোল্ডারটি খুঁজে বের করুন।
5.  এই `dist` ফোল্ডারটিকে Netlify-এর আপলোড এরিয়াতে টেনে এনে ছেড়ে দিন (drag and drop)।
6.  Netlify ফাইলগুলো আপলোড করবে এবং সফলভাবে deploy হলে আপনাকে একটি লাইভ URL দেবে।

## ধাপ ৪: Netlify-তে Deploy হওয়ার পর কী হবে?

*   আপনি একটি URL পাবেন (যেমন `https://your-site-name.netlify.app`)। এই URL-এ গেলে আপনি আপনার প্রজেক্টের ল্যান্ডিং পেজটি দেখতে পাবেন।
*   **কিন্তু মনে রাখবেন:** এই deploy করা সাইটে লগইন, ডেটাবেস অপারেশন বা অন্য কোনো ব্যাকএন্ড ফাংশনালিটি কাজ করবে না। কারণ Netlify শুধু ফ্রন্টএন্ড হোস্ট করেছে, ব্যাকএন্ড এবং ডেটাবেস আপনার লোকাল কম্পিউটারে বা অন্য কোনো সার্ভারে চলছে না।

## যদি পুরো প্রজেক্ট (ফ্রন্টএন্ড + ব্যাকএন্ড + ডেটাবেস) অনলাইনে চালাতে চান

যদি আপনি এই পুরো প্রজেক্টটি অনলাইনে চালাতে চান, তাহলে আপনাকে এমন একটি হোস্টিং সার্ভিস ব্যবহার করতে হবে যা Node.js ব্যাকএন্ড এবং MySQL ডেটাবেস সাপোর্ট করে। যেমন:

*   **Heroku** (ফ্রি টায়ার এখন নেই, কিন্তু ছোট প্রজেক্টের জন্য ভালো ছিল) [2]
*   **Render** [3]
*   **DigitalOcean** [4]
*   **AWS** (Amazon Web Services) [5]
*   **Google Cloud Platform** [6]

এই সার্ভিসগুলোতে deploy করা Netlify-এর চেয়ে কিছুটা জটিল, কারণ আপনাকে ডেটাবেস সেটআপ, এনভায়রনমেন্ট ভেরিয়েবল কনফিগারেশন এবং ব্যাকএন্ড সার্ভার চালানোর জন্য আলাদাভাবে কাজ করতে হবে।

আশা করি এই গাইডটি আপনাকে Netlify-তে প্রজেক্টের ফ্রন্টএন্ড deploy করতে সাহায্য করবে এবং Netlify-এর সীমাবদ্ধতা সম্পর্কে একটি পরিষ্কার ধারণা দেবে।
