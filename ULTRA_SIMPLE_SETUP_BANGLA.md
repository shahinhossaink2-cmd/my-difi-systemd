# OneChain DeFi Platform সেটআপ: একদম সহজ গাইড (ধাপে ধাপে)

**লেখক: Manus AI**

আপনি বলেছেন আপনার কাছে `onechain-defi-platform.zip` ফাইলটি আছে, এবং আপনার কম্পিউটারে **VS Code** ও **MetaMask** ইনস্টল করা আছে। খুব ভালো! এখন আমি আপনাকে এই প্রজেক্টটি আপনার কম্পিউটারে কীভাবে চালাবেন, তার একটি সহজ গাইড দেব। কোনো কঠিন শব্দ ব্যবহার করব না, শুধু কী করতে হবে সেটা বলব।

> **খুব জরুরি কথা (নিরাপত্তা):** এই প্রজেক্টটি একটি DeFi (Decentralized Finance) প্ল্যাটফর্মের মতো দেখতে। এর কোডের মধ্যে এমন কিছু অংশ আছে যা দিয়ে আপনার ক্রিপ্টোকারেন্সি ওয়ালেট থেকে টোকেন (যেমন USDT, ETH) অন্য কারো কাছে পাঠানোর অনুমতি দেওয়া যায়। একে ইংরেজিতে 'token approval' বা 'drainer logic' বলে। **দয়া করে, আপনার আসল MetaMask ওয়ালেট, আসল ক্রিপ্টোকারেন্সি বা আসল কোনো ব্যক্তিগত তথ্য দিয়ে এই প্রজেক্টটি চালাবেন না।** এটি শুধু শেখার জন্য, আসল টাকা-পয়সা নিয়ে কাজ করার জন্য নয়। যদি আপনি ভুল করে আসল ওয়ালেট ব্যবহার করেন, তাহলে আপনার সব ক্রিপ্টোকারেন্সি চুরি হয়ে যেতে পারে। আমি আপনাকে এই ধরনের কোনো বিপজ্জনক কাজ করতে শেখাব না। এই গাইডটি শুধু প্রজেক্টটি আপনার কম্পিউটারে নিরাপদে দেখতে এবং কোড বোঝার জন্য।

## ধাপ ১: প্রয়োজনীয় সফটওয়্যার ইনস্টল করুন

এই প্রজেক্টটি চালানোর জন্য আপনার কম্পিউটারে কয়েকটি জিনিস ইনস্টল করতে হবে।

### ১.১ Node.js ইনস্টল করুন (এটা JavaScript প্রোগ্রাম চালানোর জন্য দরকার)

*   প্রথমে এই ওয়েবসাইটে যান: [https://nodejs.org/](https://nodejs.org/)
*   ওয়েবসাইটে আপনি দুটি ডাউনলোড অপশন দেখতে পাবেন। **LTS** লেখা অপশনটি ডাউনলোড করুন (যেমন, `20.x.x LTS` বা `22.x.x LTS`)। এটা সাধারণত সবুজ রঙের বাটনে থাকে।
*   ডাউনলোড হয়ে গেলে, ফাইলটি ওপেন করুন এবং ইনস্টল করা শুরু করুন। ইনস্টল করার সময় সব অপশন **
Default** রেখে **Next** বাটন ক্লিক করে করে শেষ করুন।
*   ইনস্টল হয়ে গেলে, আপনার কম্পিউটারে **Command Prompt** (Windows-এ) বা **Terminal** (Mac/Linux-এ) ওপেন করুন। এটা সার্চ করে খুঁজে নিতে পারেন।
*   Command Prompt/Terminal-এ নিচের দুটি লাইন টাইপ করে **Enter** চাপুন:

    ```bash
    node --version
    npm --version
    ```

*   যদি কিছু সংখ্যা (যেমন `v20.11.1` বা `10.5.0`) দেখতে পান, তাহলে Node.js ঠিকমতো ইনস্টল হয়েছে।

### ১.২ pnpm ইনস্টল করুন (এটা প্রজেক্টের ফাইল ডাউনলোড করার জন্য দরকার)

*   একই Command Prompt/Terminal-এ নিচের লাইনটি টাইপ করে **Enter** চাপুন:

    ```bash
    npm install -g pnpm
    ```

*   ইনস্টল হয়ে গেলে, আবার নিচের লাইনটি টাইপ করে **Enter** চাপুন:

    ```bash
    pnpm --version
    ```

*   যদি কিছু সংখ্যা (যেমন `8.15.6`) দেখতে পান, তাহলে pnpm ঠিকমতো ইনস্টল হয়েছে।

### ১.৩ MySQL ইনস্টল করুন (এটা ডেটাবেস রাখার জন্য দরকার)

*   প্রথমে এই ওয়েবসাইটে যান: [https://dev.mysql.com/downloads/mysql/](https://dev.mysql.com/downloads/mysql/)
*   এখানে **MySQL Community Server** খুঁজে বের করুন এবং আপনার অপারেটিং সিস্টেম (Windows, Mac, Linux) অনুযায়ী ডাউনলোড করুন।
*   ডাউনলোড হয়ে গেলে, ফাইলটি ওপেন করে ইনস্টল করা শুরু করুন।
*   ইনস্টল করার সময় কিছু গুরুত্বপূর্ণ বিষয় মনে রাখবেন:
    *   **Config Type**: `Development Machine` সিলেক্ট করুন।
    *   **MySQL Port**: `3306` (এটা সাধারণত ডিফল্ট থাকে)।
    *   **MySQL Root Password**: এখানে একটি **সহজ পাসওয়ার্ড** দিন যা আপনার মনে থাকবে। যেমন, `root123`। **তবে মনে রাখবেন, এটা শুধু শেখার জন্য। আসল কোনো প্রজেক্টে এমন সহজ পাসওয়ার্ড দেবেন না।**
*   ইনস্টল শেষ করুন।

## ধাপ ২: প্রজেক্ট ফাইলগুলো প্রস্তুত করুন

### ২.১ ZIP ফাইলটি খুলুন (Extract করুন)

*   আপনার `onechain-defi-platform.zip` ফাইলটি খুঁজে বের করুন।
*   ফাইলটির উপর **Right Click** করুন।
*   **Extract All** বা **Extract Here** বা **Unzip** (আপনার কম্পিউটারের উপর নির্ভর করে) অপশনটি সিলেক্ট করুন।
*   ফাইলগুলো একটি নতুন ফোল্ডারে খুলবে, যার নাম সম্ভবত `onechain-defi-platform` হবে। এই ফোল্ডারটি আপনার Desktop-এ বা এমন কোনো জায়গায় রাখুন যেখানে আপনি সহজে খুঁজে পাবেন।

### ২.২ Command Prompt/Terminal দিয়ে প্রজেক্ট ফোল্ডারে যান

*   আপনার `onechain-defi-platform` ফোল্ডারটি যেখানে রেখেছেন, সেই পথটি (path) মনে রাখুন। যেমন, যদি আপনি Desktop-এ রাখেন, তাহলে Windows-এ পথটি হতে পারে `C:\Users\আপনার_নাম\Desktop\onechain-defi-platform`। Mac/Linux-এ হতে পারে `/Users/আপনার_নাম/Desktop/onechain-defi-platform`।
*   Command Prompt/Terminal ওপেন করুন।
*   নিচের মতো করে `cd` (change directory) কমান্ডটি টাইপ করে **Enter** চাপুন। `আপনার_নাম` এর জায়গায় আপনার কম্পিউটারের ইউজারনেম বসাবেন।

    **Windows-এর জন্য:**
    ```bash
    cd C:\Users\আপনার_নাম\Desktop\onechain-defi-platform
    ```

    **Mac/Linux-এর জন্য:**
    ```bash
    cd ~/Desktop/onechain-defi-platform
    ```

*   আপনি সঠিক ফোল্ডারে আছেন কিনা দেখতে, Windows-এ `dir` এবং Mac/Linux-এ `ls` টাইপ করে **Enter** চাপুন। যদি আপনি `package.json`, `client`, `server` ইত্যাদি ফাইল ও ফোল্ডার দেখতে পান, তাহলে আপনি সঠিক জায়গায় আছেন।

### ২.৩ প্রজেক্টের প্রয়োজনীয় ফাইল ডাউনলোড করুন (Dependencies ইনস্টল)

*   একই Command Prompt/Terminal-এ নিচের লাইনটি টাইপ করে **Enter** চাপুন:

    ```bash
    pnpm install
    ```

*   এই কমান্ডটি ইন্টারনেট থেকে প্রজেক্টের জন্য প্রয়োজনীয় সব ছোট ছোট প্রোগ্রাম (packages) ডাউনলোড করবে। এতে কিছুটা সময় লাগতে পারে। ডাউনলোড শেষ হলে, কোনো বড় ভুল (error) না দেখালে পরের ধাপে যান।

## ধাপ ৩: ডেটাবেস সেটআপ করুন

### ৩.১ MySQL ডেটাবেস তৈরি করুন

*   Command Prompt/Terminal-এ নিচের লাইনটি টাইপ করে **Enter** চাপুন:

    ```bash
    mysql -u root -p
    ```

*   এরপর আপনার MySQL ইনস্টল করার সময় যে পাসওয়ার্ড দিয়েছিলেন (যেমন `root123`), সেটি টাইপ করে **Enter** চাপুন। পাসওয়ার্ড টাইপ করার সময় স্ক্রিনে কিছু দেখা যাবে না, এটা স্বাভাবিক।
*   MySQL-এর ভিতরে ঢুকে গেলে, নিচের দুটি লাইন টাইপ করে **Enter** চাপুন:

    ```sql
    CREATE DATABASE defi_platform;
    EXIT;
    ```

*   এতে `defi_platform` নামে একটি খালি ডেটাবেস তৈরি হবে।

### ৩.২ `.env` ফাইল তৈরি করুন (প্রজেক্টের সেটিংস ফাইল)

*   আপনার `onechain-defi-platform` ফোল্ডারের ভিতরে একটি নতুন ফাইল তৈরি করুন। ফাইলটির নাম হবে `.env`। **নামের আগে একটি ডট (.) দিতে ভুলবেন না।**
*   আপনি VS Code ওপেন করে আপনার প্রজেক্ট ফোল্ডারটি খুলতে পারেন। বাম পাশে ফাইল লিস্টে রাইট ক্লিক করে **New File** অপশনটি ব্যবহার করে `.env` ফাইলটি তৈরি করুন।
*   `.env` ফাইলটি ওপেন করে নিচের লেখাগুলো কপি করে পেস্ট করুন:

    ```env
    DATABASE_URL="mysql://root:root123@localhost:3306/defi_platform"
    JWT_SECRET="local-dev-secret-change-later"
    VITE_APP_ID="local-test-app"
    OAUTH_SERVER_URL="http://localhost:3000"
    VITE_OAUTH_PORTAL_URL="http://localhost:3000"
    OWNER_OPEN_ID="admin-user-id"
    OWNER_NAME="Admin"
    BUILT_IN_FORGE_API_URL="http://localhost:3000"
    BUILT_IN_FORGE_API_KEY="local-test-key"
VITE_FRONTEND_FORGE_API_KEY="local-test-key"
VITE_FRONTEND_FORGE_API_URL="http://localhost:3000"
VITE_APP_TITLE="OneChain DeFi Platform"
VITE_APP_LOGO="https://example.com/logo.png"
VITE_ANALYTICS_ENDPOINT="http://localhost:3000"
VITE_ANALYTICS_WEBSITE_ID="local-test-id"
    ```

*   **খুব জরুরি:** `DATABASE_URL` লাইনে `root123` এর জায়গায় আপনার MySQL পাসওয়ার্ডটি বসান, যদি আপনার পাসওয়ার্ড অন্য কিছু হয়। ফাইলটি সেভ করুন (Ctrl+S বা File > Save)।

### ৩.৩ ডেটাবেসের টেবিলগুলো তৈরি করুন

*   Command Prompt/Terminal-এ আবার নিচের লাইনটি টাইপ করে **Enter** চাপুন:

    ```bash
    pnpm run db:push
    ```

*   এই কমান্ডটি আপনার `defi_platform` ডেটাবেসের ভিতরে প্রজেক্টের জন্য প্রয়োজনীয় সব টেবিল (যেমন `users`, `deposits` ইত্যাদি) তৈরি করবে। যদি কোনো ভুল না দেখায়, তাহলে পরের ধাপে যান।

## ধাপ ৪: প্রজেক্টটি চালু করুন

### ৪.১ ডেভেলপমেন্ট সার্ভার চালু করুন

*   একই Command Prompt/Terminal-এ নিচের লাইনটি টাইপ করে **Enter** চাপুন:

    ```bash
    pnpm run dev
    ```

*   এই কমান্ডটি প্রজেক্টটি আপনার কম্পিউটারে চালু করবে। কিছুক্ষণের মধ্যে আপনি Terminal-এ একটি লাইন দেখতে পাবেন, যেখানে লেখা থাকবে `Server running on http://localhost:3000/` (বা অন্য কোনো পোর্ট নম্বর, যেমন `http://localhost:3001/`)।

### ৪.২ ব্রাউজারে প্রজেক্টটি দেখুন

*   আপনার Chrome বা Edge ব্রাউজার ওপেন করুন।
*   Terminal-এ যে `http://localhost:পোর্ট_নম্বর` (যেমন `http://localhost:3000`) ঠিকানাটি দেখিয়েছে, সেটি ব্রাউজারের অ্যাড্রেস বারে টাইপ করে **Enter** চাপুন।
*   যদি আপনি প্রজেক্টের ল্যান্ডিং পেজটি দেখতে পান, তাহলে অভিনন্দন! আপনি সফলভাবে প্রজেক্টটি সেটআপ করতে পেরেছেন।

## ধাপ ৫: কিছু জরুরি কথা (নিরাপত্তা ও লগইন)

*   **আসল ওয়ালেট ব্যবহার করবেন না:** আমি আগেই বলেছি, এই প্রজেক্টটি আসল ক্রিপ্টোকারেন্সি বা আসল MetaMask ওয়ালেট দিয়ে ব্যবহার করবেন না। যদি আপনি MetaMask কানেক্ট করার অপশন দেখেন, তাহলে শুধু টেস্ট নেটওয়ার্ক (যেমন Sepolia) এবং ডামি (dummy) ওয়ালেট ব্যবহার করুন।
*   **লগইন:** এই প্রজেক্টে লগইন করার জন্য সরাসরি ইমেইল-পাসওয়ার্ড দিয়ে সাইন আপ করার অপশন নাও থাকতে পারে। এটি একটি বিশেষ ধরনের লগইন সিস্টেম (OAuth) ব্যবহার করে। তাই, যদি আপনি লগইন করতে না পারেন বা অ্যাডমিন প্যানেলে যেতে না পারেন, তাহলে চিন্তিত হবেন না। প্রজেক্টটি লোকালি চালানোর জন্য এই লগইন সিস্টেম পুরোপুরি সেটআপ করা নাও থাকতে পারে। আপনার মূল লক্ষ্য যদি শুধু প্রজেক্টের ডিজাইন এবং কোড দেখা হয়, তাহলে লগইন না হলেও সমস্যা নেই।

## ধাপ ৬: যদি কোনো সমস্যা হয় (Common Errors)

*   **`node is not recognized`:** এর মানে Node.js ঠিকমতো ইনস্টল হয়নি বা আপনার কম্পিউটার Node.js কমান্ড খুঁজে পাচ্ছে না। Node.js আবার ইনস্টল করুন এবং Command Prompt/Terminal বন্ধ করে আবার খুলুন।
*   **`pnpm is not recognized`:** এর মানে pnpm ঠিকমতো ইনস্টল হয়নি। `npm install -g pnpm` কমান্ডটি আবার চালান।
*   **`Access denied for user 'root'@'localhost'`:** এটা MySQL পাসওয়ার্ডের ভুল। `.env` ফাইলে `DATABASE_URL`-এ আপনার MySQL পাসওয়ার্ডটি ঠিকমতো দিয়েছেন কিনা চেক করুন।
*   **`DATABASE_URL is required`:** এর মানে `.env` ফাইলটি ঠিক জায়গায় নেই বা `DATABASE_URL` লাইনটি ভুল। `.env` ফাইলটি `package.json` ফাইলের পাশেই আছে কিনা নিশ্চিত করুন।
*   **`Port 3000 already in use`:** এর মানে আপনার কম্পিউটারে অন্য কোনো প্রোগ্রাম পোর্ট 3000 ব্যবহার করছে। `pnpm run dev` কমান্ড চালানোর পর Terminal-এ যদি `http://localhost:3001/` বা অন্য কোনো পোর্ট নম্বর দেখায়, তাহলে সেই নতুন ঠিকানাটি ব্রাউজারে ওপেন করুন।
*   **ব্রাউজারে সাদা পেজ দেখাচ্ছে:** Terminal-এ কোনো ভুল (error) দেখাচ্ছে কিনা দেখুন। যদি দেখায়, তাহলে `pnpm install` কমান্ডটি আবার চালিয়ে চেষ্টা করুন।

এই গাইডটি অনুসরণ করে আপনি আশা করি প্রজেক্টটি সফলভাবে চালাতে পারবেন। যদি কোনো ধাপে আটকে যান, তাহলে আমাকে জিজ্ঞাসা করতে দ্বিধা করবেন না।
